Step 1: Run Locally (5 minutes)
  $ cd web-dashboard
  $ npm install
  $ npm run dev
  → Open http://localhost:5173

Step 2: Build for Production (10 minutes)
  $ npm run build
  $ npm run preview

Step 3: Deploy on Render (15 minutes)
  1. Push to GitHub
  2. Create Render account
  3. Connect GitHub repo
  4. Set build command: npm --prefix web-dashboard run build
  5. Set start command: npm --prefix web-dashboard run preview
  6. Add environment variables
  7. Deploy

Step 4: Connect Backend
  1. Update VITE_API_BASE_URL in .env
  2. Deploy backend API
  3. Verify API health check
  4. Test data flow
